## How to Describe Pictures at B1 Level

Picture description is a key part of the OEIF B1 speaking exam. This page provides a clear technique and multiple examples across different contexts.

### The Picture Description Pattern

1. **Overview**: What is happening in general?
2. **Details**: Who, where, what are they doing?
3. **Speculation**: What might happen next or why?

## Common Mistakes to Avoid

- Don't just list objects; describe actions and relationships
- Use present tense for what you see, speculation for interpretation
- Connect details logically
